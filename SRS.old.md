# Software Requirements Specification (SRS)
**โปรเจกต์:** ระบบสั่งซื้อสินค้าและหน้าเว็บร้านกาแฟ "Brown Coffee"

## 1. ภาพรวมระบบ
ระบบ "Brown Coffee" เป็นเว็บแอปพลิเคชันที่มีหน้า Landing Page โทนสีน้ำตาล มินิมอล อบอุ่น เพื่อนำเสนอร้านและเมนูเด่น พร้อมด้วยระบบตะกร้าสินค้าแบบ E-commerce เต็มรูปแบบ เพื่อให้ลูกค้าสามารถเลือกซื้อกาแฟแบบ Guest และเลือกชำระเงินปลายทาง (COD) หรือรับที่ร้านได้ทันที นอกจากนี้ยังมีระบบ Admin Dashboard สำหรับเจ้าของร้านเพื่อเข้ามาจัดการและตรวจสอบคำสั่งซื้อที่เข้ามาในระบบ โดยออกแบบมาให้รันและทดสอบบน Localhost

## 2. ผู้ใช้
*   **Customer (ลูกค้าทั่วไป):** ผู้ที่เข้ามาเยี่ยมชมเว็บไซต์ ดูเมนูกาแฟ เพิ่มสินค้าลงตะกร้า และทำการสั่งซื้อสินค้าแบบไม่ต้องสมัครสมาชิก
*   **Admin (เจ้าของร้าน):** ผู้ที่ล็อกอินเข้าสู่ระบบหลังบ้านเพื่อดูรายการคำสั่งซื้อ เปลี่ยนแปลงสถานะออเดอร์ (เช่น รอจัดส่ง, จัดส่งแล้ว)

## 3. ฟีเจอร์ "ต้องมี" (Must Have)
*   **Customer Facing:**
    *   หน้า Landing Page สไตล์มินิมอล โทนน้ำตาล อบอุ่น (Responsive)
    *   ส่วน Hero section น่าสนใจ
    *   แสดงเมนูแนะนำ 3 อย่าง (ข้อมูลจำลอง Mockup) พร้อมปุ่ม "สั่งซื้อ"
    *   ระบบตะกร้าสินค้า (Add to Cart, View Cart)
    *   หน้า Checkout สำหรับกรอกข้อมูลสั่งซื้อแบบ Guest (ไม่ต้อง Login)
    *   ตัวเลือกการรับสินค้า: เก็บเงินปลายทาง (COD) หรือ รับที่ร้าน
*   **Admin Facing:**
    *   ระบบล็อกอินสำหรับ Admin
    *   หน้าระบบหลังบ้าน (Dashboard) แสดงรายการคำสั่งซื้อทั้งหมด
    *   ฟังก์ชันอัปเดตสถานะออเดอร์ (รอจัดส่ง / เสร็จสิ้น)

## 4. ฟีเจอร์ "น่ามี" (Nice to Have)
*   ระบบสมาชิกสำหรับลูกค้า (ลงทะเบียน/เข้าสู่ระบบ) เพื่อเก็บประวัติการสั่งซื้อ
*   ระบบชำระเงินออนไลน์ (ตัดบัตรเครดิต, สแกน QR Code PromptPay)
*   การส่งแจ้งเตือนออเดอร์ใหม่เข้า LINE OA หรือ Email ของเจ้าของร้าน
*   ระบบจัดการเมนู (เพิ่ม/ลบ/แก้ไข รายการเครื่องดื่ม) บนระบบหลังบ้าน

## 5. ข้อมูลที่ต้องเก็บ (Data Requirements)

| กลุ่มข้อมูล | ฟิลด์ที่ต้องจัดเก็บ | หมายเหตุ |
| :--- | :--- | :--- |
| **ข้อมูลลูกค้า (Customer)** | ชื่อ-นามสกุล, เบอร์โทรศัพท์, ที่อยู่จัดส่ง | ที่อยู่ใช้ในกรณีเลือกชำระเงินปลายทาง (COD) |
| **ข้อมูลออเดอร์ (Order)** | รายการเมนูที่สั่ง, จำนวน, ราคารวม, รูปแบบการรับสินค้า, สถานะออเดอร์ | สถานะ: รอจัดส่ง / เสร็จสิ้น |
